# Checklist de coleta do cliente — RAIOX V1

Preencher apenas o que estiver disponível. Informações sensíveis e dados pessoais devem ser ocultados sempre que não forem necessários para a análise.

## Dados básicos

- [ ] Nome da empresa.
- [ ] Responsável pelo envio.
- [ ] Produto ou serviço principal.
- [ ] Público-alvo.
- [ ] Região atendida.
- [ ] Meta comercial atual.

## Site

- [ ] URL principal.
- [ ] Páginas de produto ou serviço.
- [ ] Landing pages ativas.
- [ ] Formulários e CTAs existentes.
- [ ] Dados disponíveis de visitas e conversões.
- [ ] Principais problemas percebidos no site.

## Instagram

- [ ] Link do perfil.
- [ ] Frequência de publicação.
- [ ] Tipos de conteúdo utilizados.
- [ ] Forma de entrada dos leads.
- [ ] Métricas disponíveis dos últimos 30 dias.
- [ ] Exemplos de posts que geraram conversas.

## Google Business

- [ ] Link ou nome exato do perfil.
- [ ] Categoria principal e categorias secundárias.
- [ ] Serviços cadastrados.
- [ ] Quantidade e nota das avaliações.
- [ ] Frequência de respostas e publicações.
- [ ] Acesso às métricas disponíveis.

## WhatsApp

- [ ] Número ou canal usado pela empresa.
- [ ] Quantidade de atendentes.
- [ ] Horário de atendimento.
- [ ] Tempo médio de primeira resposta.
- [ ] Mensagem inicial e respostas rápidas.
- [ ] Etiquetas e organização atual.
- [ ] Amostra de conversas com dados pessoais ocultados.

## Anúncios

- [ ] Plataformas utilizadas.
- [ ] Objetivo das campanhas.
- [ ] Investimento médio mensal.
- [ ] Volume de leads gerados.
- [ ] Custo por lead, se disponível.
- [ ] Páginas e mensagens de destino.

## Atendimento

- [ ] Quem recebe o primeiro contato.
- [ ] Roteiro ou padrão de abordagem.
- [ ] Perguntas de qualificação.
- [ ] Principais dúvidas dos leads.
- [ ] Principais objeções.
- [ ] Critérios para encaminhar ou descartar oportunidades.

## Funil

- [ ] Etapas atuais do processo comercial.
- [ ] Critério de avanço entre etapas.
- [ ] Responsável por cada etapa.
- [ ] Volume aproximado em cada etapa.
- [ ] Taxas de conversão disponíveis.
- [ ] Motivos de perda registrados.

## Follow-up

- [ ] Quantidade de tentativas realizadas.
- [ ] Intervalo entre tentativas.
- [ ] Canais utilizados.
- [ ] Modelos de mensagem atuais.
- [ ] Regra para encerrar o acompanhamento.
- [ ] Percentual de leads sem segunda tentativa, se disponível.

## CRM

- [ ] Ferramenta, planilha ou método utilizado.
- [ ] Campos obrigatórios.
- [ ] Etapas configuradas.
- [ ] Responsáveis pelo preenchimento.
- [ ] Automação existente.
- [ ] Qualidade e frequência de atualização.

## Métricas

- [ ] Leads recebidos por mês.
- [ ] Origem dos leads.
- [ ] Tempo médio de resposta.
- [ ] Taxa de qualificação.
- [ ] Taxa de proposta.
- [ ] Taxa de fechamento.
- [ ] Ticket médio.
- [ ] Ciclo médio de venda.
- [ ] Motivos de perda.
- [ ] Meta atual.

## Confirmação final

- [ ] Os dados pessoais desnecessários foram ocultados.
- [ ] Os links possuem acesso liberado.
- [ ] As métricas informam o período analisado.
- [ ] A amostra representa a operação real.
- [ ] O contexto e os principais gargalos foram descritos.
